## Что такое CI/CD?

**CI/CD** — это практика автоматизации процессов разработки программного обеспечения.

### Continuous Integration (CI) — Непрерывная интеграция

- Разработчики регулярно пушат код в общий репозиторий
- При каждом пуше автоматически запускаются сборка и тесты

**Преимущества CI:**
- Раннее обнаружение ошибок
- Быстрая обратная связь

### Continuous Deployment (CD) — Непрерывное развертывание

Каждое успешное изменение автоматически попадает на сервер.

## Как CI/CD работает в моём проекте

1. **CI**: GitHub Actions проверяет сборку сайта
2. **CD**: При успехе сайт публикуется на GitHub Pages

## Популярные CI/CD инструменты

- **GitHub Actions** — бесплатен для открытых репозиториев
- **GitLab CI** — встроен в GitLab
- **Jenkins** — мощный, но сложный

## Заключение

CI/CD ускоряет разработку и повышает качество кода.
